PRAGMATA Script & Mod Framework

Extensible environment for custom Pragmata modifications

PRAGMATA Script & Mod Framework is a lightweight integration layer designed to enable user-created extensions for Pragmata. It provides support for Lua scripting and offers a structured way to load and manage community-made modifications.

Features
Integrated and adapted REFramework build for Pragmata
Lua scripting support
Automatic mod initialization system
Developer console and debugging tools
Compatibility with external graphics presets (e.g. RenoDX)
Simple setup and minimal configuration
Installation
Download the archive: Pragmata Script & Mod Framework.zip
Extract the contents into your game directory
Make sure you are using the latest version of Pragmata
Optionally install additional graphical enhancements
Troubleshooting
Issue	Solution
Framework does not initialize	Ensure the game is running in DirectX 12 mode
Mods are not loading	Place mod files inside the reframework/autorun folder
Game crashes on startup	Try running the game as administrator
Console not visible	Enable it in the framework settings
Antivirus warning	Add the folder to exclusions
